# 🪟 Imzo Windows — Sales & Logistics Analytics Dashboard

## 📊 Overview
Power BI dashboard analyzing sales performance, regional
plan execution, delivery status, and monthly growth dynamics
for Imzo — a windows and construction materials company
operating across Central Asia.

## 🛠️ Tools Used
- Power BI Desktop
- DAX (Data Analysis Expressions)
- Excel / CSV data source

## 📌 Key Metrics
- Total Sales (KLAES): $1,248.27K
- Total Debt: $509.03K
- Average Discount: 51%
- Total Area: 13.22K m²
- Total Quantity: 1,560 pcs
- Growth Rate: 11%
- Plan Execution: 53%
- Overdue Orders: 12
- Avg Overdue Days: 8.25
- Total Debtors: 35

## 📈 Dashboard Features
- Regional plan execution comparison (2025 actual vs plan)
- Sales volume by category — PVC / Aluminium / Glass (donut chart)
- Delivery status breakdown (Delivered vs Not Ready)
- Monthly sales dynamics with YoY growth % (combo chart)
- KPI cards (Sales, Debt, Discount, Area, Quantity, Growth)
- Plan execution gauge (0–100%)
- Overdue orders & debtor tracking
- Region filter (dropdown)

## 🌍 Regions Covered
Tajikistan • Kyrgyzstan • Russia • Afghanistan • Kazakhstan
